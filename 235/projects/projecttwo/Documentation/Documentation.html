Concept

A simple guessing game game that leverages the Jikan API to allow the player to guess which result has a higher value of a chosen parameter than the other.

Process

Most of the process was trial and error if I wasn't procrastinating. I read a lot of documentation to get the more complex functionality, but most of the stuff I did I just freeballed because it was pretty simple, just boolean flipping and updating variables.

For specifics on the API requests, they are very simple. It uses 3 requests. The first request gets all of the anime based on the users search parameters. This request, however, is paginated, so from there I use the number of pages to get a random page number and pick a random anime off of that page. I made sure the same number could not be gotten twice. Everything is randomized, including the type of show/book that is picked. NSFW results are filtered out in multiple ways.
